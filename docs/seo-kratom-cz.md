# SEO-анализ: продажа кратома в Чехии

Дата анализа: 2026-04-08

## Контекст ниши

С 12.11.2025 kratom и kratom extrakt в Чехии включены в список psychomodulační látky. Для e-shop продаж нужен допуск Министерства здравоохранения и рабочая схема проверки возраста через квалифицированную электронную идентификацию NIA. Поэтому конкурентная выдача сместилась от простого "koupit kratom" к кластерам доверия: licencovaný e-shop, povolení MZ, ověření věku Adulto/NIA, laboratorní testy, kamenná prodejna и rychlé doručení.

## Позиционирование VivaDzen

Основная связка для SEO: "certifikovaný kratom pro Českou republiku" + "kamenná prodejna Praha" + "ověření věku ADULTO/NIA" + "laboratorní kontrola a certifikáty" + "doručení po ČR" + "vícejazyčný servis".

Что избегать:
- медицинских обещаний и фраз "léčí", "pomáhá na bolest/úzkost/depresi", "odvykání opioidů";
- "bez ověření věku", "prodej bez Adulto", "pro mladší 18 let";
- city landing pages для городов, где нет реальной точки/доставки;
- переспама "nejlevnější" без ценового позиционирования.

## SEO-аудит и внедренные рекомендации

- Метатеги централизовать в `lang/{locale}.yaml` через `seo.pages.*`.
- Для товара и статьи использовать шаблоны с `{name}` и `{title}` как fallback, если API не вернул `seo.meta_title`/`seo.meta_description`.
- Для всех страниц отдавать `description`, `og:title`, `og:description`, `og:type`, `twitter:*`.
- Для checkout-flow ставить `noindex, nofollow` и исключать из robots/sitemap.
- Для `/catalog?page=2` и `/blog?page=2` оставлять self-canonical с query `?page=N`, чтобы не каноникалить все пагинированные страницы в первую.
- Не индексировать URL с tracking query (`gclid`, `fbclid`, `utm_*`).
- На главной/landing layout добавить WebSite/WebPage schema, как уже было в default layout.
